## Problem Statement

Machines breakdowns can lead to lost productivity, safety hazards, and financial losses.

## Solution Overview

ForwardX is a machine maintenance app designed to prevent breakdowns by tracking machine status and scheduling maintenance tasks.
<img width="865" alt="image" src="https://github.com/Harshitanand11/ForwardX/assets/83498313/90fed680-d1e3-4e8c-b68b-fdaf0dfd768f">


## Implementation

- Developed using Flutter and Dart
- Designed in Figma
- Integrated with Firebase for real-time updates
- Utilizes Google Auth for login
  <img width="866" alt="image" src="https://github.com/Harshitanand11/ForwardX/assets/83498313/7239702b-7354-4687-9b1a-a57ab8dc03dc">


## Features

- **Login Screen**: Sign in with Google account
- **Tabbed Screen**: Dashboard, Machine List, Notification List, Location List
- **Machine Details Page**: View machine status, activate/deactivate machines, view messages
- **Image Upload**: Upload images of faulty machines
- **Download Files**: Access system-related files
- **Bilingual Support**: Available in English and Hindi
- **Log out / Delete account**: Easily manage account settings
